Selam! Ben Kutay. Kutay Altuntaş. Ankara'nın Altındağ İlçesinde Doğdum ve Büyüdüm. 24 Şubat 2003 Doğumluyum. Gazi Üniversitesi Uygulamalı Bilimler Fakültesi Yönetim Bilişim Sistemleri Bölümü 2. sınıf öğrencisiyim. Amacım kendimi yazılım alanında geliştirmek. Web programlama ve Yapay Zekaya ilgim var. Web programlama konusunda HTML, CSS ve JavaScript gibi dilleri kullanabiliyorum. Yapay zeka konusunda ise Python ile TensorFlow ve Keras gibi kütüphaneleri öğreniyorum. Gelecekte, web siteleri ve mobil uygulamalar geliştirmek ve yapay zeka projelerine katılmak istiyorum.
